# 八字排盘 H5 应用

这是一个基于 Vue3 + Nuxt3 开发的移动端 H5 应用，用于实现八字排盘功能。

## 功能

- 用户输入公历生日时，自动计算八字、大运、流年。
- 展示四柱、十神、地支藏干等信息。
- 支持通过链接分享和查询历史排盘记录。

## 技术栈

- **前端**: Vue3, Nuxt3, Pinia, UnoCSS, Avola UI
- **后端**: Nuxt3 Server API
- **数据库**: Prisma, SQLite
- **构建工具**: pnpm
